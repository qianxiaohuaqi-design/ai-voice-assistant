🤖 智能语音助手说明书
🔑 1. 如何配置 API Key 才能开始？

点击左下角侧边栏的 “系统设置” 按钮，拉到最底部的 “开发者 API 配置”：

  • AI 大模型 Key: 填写您的 API Key。本助手不限制模型渠道，全面支持 DeepSeek, OpenAI, Claude, Gemini, Grok 等各种大模型或第三方中转接口的 API Key。若使用代理或中转接口（例如 OhMyGPT），可配合在下方填写自定义的代理地址 (Base URL)。
  • ElevenLabs Key: 填写您的语音合成 API Key，用于生成精美的音色。
🎙️ 2. 怎么添加和切换语音声音？

点击左下角侧边栏的 “系统设置”，在中间的 “声音库管理” 卡片中：

  • 在输入框中填入自定义声音名称以及您在 ElevenLabs 官方后台克隆或获取的 Voice ID。
  • 点击 “添加新声音” 成功后，即可在上方 “选择语音声音” 下拉菜单中一键切换！
🛡️ 3. 右上角按键功能与省额度秘籍
  • 语音/静音按钮 (): 一键开启/关闭语音。当切换为“纯文字模式”时，后端会直接跳过 ElevenLabs 语音合成接口，发送请求时不消耗 ElevenLabs 额度，非常适合打字或测试时使用。
  • 星云按钮 (): 一键在“传统聊天面板”和“3D 粒子星云球”间切换。在星云模式下，AI 回复会以柔和的“原地浮现”动态字幕展示，并且字随音走，未来科技感十足。
